## ADDED Requirements

### Requirement: Side Panel Access
The extension SHALL provide a Side Panel interface accessible from the Chrome toolbar while navigating Google Maps.

#### Scenario: Opening the Side Panel
- **WHEN** the user clicks the extension icon while on Google Maps
- **THEN** the Side Panel opens displaying the configuration settings

### Requirement: Side Panel Settings View
The Side Panel SHALL render a basic UI allowing the user to view and adjust scraping settings.

#### Scenario: Viewing settings
- **WHEN** the Side Panel is visible
- **THEN** it displays mock or placeholder controls for categories and limits
